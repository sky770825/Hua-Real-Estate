# 🔐 手動登錄 Cloudflare 步驟

## 為什麼需要手動登錄？

`wrangler login` 需要打開瀏覽器進行 OAuth 授權，在自動化環境中可能無法完成。

## 📋 步驟：

### 1. 打開終端（Terminal）

在您的 Mac 上打開終端應用程序。

### 2. 進入項目目錄

```bash
cd "/Users/caijunchang/Desktop/程式專案資料夾/Hua-Real-Estate-華地產網站"
```

### 3. 運行登錄命令

```bash
wrangler login
```

### 4. 完成瀏覽器授權

- 命令會自動打開瀏覽器
- 登錄您的 Cloudflare 帳戶
- 點擊 "Allow" 授權 Wrangler 訪問

### 5. 確認登錄成功

登錄完成後，終端會顯示：
```
✅ Successfully logged in.
```

---

## ✅ 登錄完成後

運行部署命令：
```bash
wrangler deploy
```

部署成功後，複製 Worker URL 告訴我，我會幫您配置！
